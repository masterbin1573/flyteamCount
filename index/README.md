# 飞天蒜子统计 - 静态首页

这是飞天蒜子统计项目的静态首页文件夹。

## 📁 文件说明

- `index.html` - 主页面，包含完整的产品介绍、使用说明和功能演示

## 🚀 部署说明

这个文件夹可以独立部署到任何静态网站托管服务：

- **Vercel** - 直接连接 GitHub 仓库部署
- **Netlify** - 拖拽文件夹部署
- **GitHub Pages** - 设置为根目录部署
- **CDN** - 上传到任何支持静态文件的CDN

## ✨ 特性

- 纯静态HTML页面，加载速度极快
- 响应式设计，完美支持移动端
- 集成了飞天蒜子统计和点赞功能演示
- 包含完整的使用文档和代码示例
- 无需任何框架依赖
